**--Hogwarts--
**
This is a student project, codded on Java (Spring Boot).
Contains a backend component, including a database PostgreSQL. 
Allows adding students, departments, and avatars. 
Basic CRUD queries and many additional ones (for training purposes) are implemented.

How to use?
The project runs in IntelliJ IDEA and is accessible via Swagger at http://localhost:8080/swagger-ui/index.html.
A ready-to-use Docker image of the application is also available.
If you need it, I can send you the image file. My email: evis10@yandex.ru.
Simply load it into Docker and run the container.
